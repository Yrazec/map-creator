# Map Creator

Create 2D map with treasure.

**The map ALWAYS has one entrance, one exit, and an existing path from one to the other!**

# Late Execution Features

1. Print map to console.
2. Save map to file.

# Usage

```commandline
python main.py
```

# Requirements

- Python 3.11.9
- numpy==2.3.2
